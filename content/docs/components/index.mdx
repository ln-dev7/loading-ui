These pages document **props**, **installation**, and **examples** for each piece. Start from a component below when you know what you need, or read [Usage](/docs/usage) first if you are setting up the project.

## Catalog

| Component | When to use it |
| --- | --- |
| [Ring](/docs/components/ring) | General-purpose indeterminate loading, inline actions, and centered fallbacks |
| [Spokes](/docs/components/spokes) | Lighter segmented loading indicators for compact UI and toolbars |

As more components are documented, they will appear in this table and in the sidebar.

## Conventions

- Examples assume **TypeScript** and **Tailwind** unless stated otherwise.
- Styling uses design tokens from your app; class names in snippets are illustrative.
- For breaking changes, check the project changelog or release notes when available.
